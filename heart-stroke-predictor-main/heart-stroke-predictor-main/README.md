# ❤️ Heart Stroke Prediction Web Application

## 📌 Project Overview

The Heart Stroke Prediction project is a Machine Learning based web application developed to predict the risk of heart disease/stroke using patient health information. The application uses a trained K-Nearest Neighbors (KNN) model to analyze various medical parameters and provide a prediction result.

This project was built using Python, Scikit-learn, Pandas, and Streamlit, and deployed online using Streamlit Cloud.

## 🎯 Objective

The main objective of this project is to create a simple and interactive healthcare prediction system that can help users understand the possible risk of heart disease based on medical attributes.

The application takes multiple health-related inputs from the user and predicts whether the person has a high or low risk of heart disease.

---

## 🧠 Machine Learning Model Used

### ✅ K-Nearest Neighbors (KNN)

This project uses the KNN Classification Algorithm.

KNN works by:

* finding the nearest data points to the input data
* comparing similarities between patients
* predicting the output based on neighboring data points

KNN was chosen because:

* it is simple and effective
* performs well on classification problems
* easy to understand and implement

---

## 📊 Features Used for Prediction

The model uses the following medical features:

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol Level
* Fasting Blood Sugar
* Resting ECG
* Maximum Heart Rate
* Exercise-Induced Angina
* Old Peak
* ST Slope
